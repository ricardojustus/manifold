# overlays/<name>/rules/

Project-specific rule files (one `.md` each). Installed to `.claude/rules/` alongside the
core rules. Use for binding project directives that don't generalize into core (e.g. an
agent-spawn vocabulary contract). This `README.md` is a placeholder — the installer skips it.
Delete it once you add real rule files, or leave it; either way it is not installed.
